# PhpOne
# Rito backend

## Setup
1. Add ```127.0.0.20 api.getrito.com.loc ``` in ```/etc/hosts``` 
1. Run on the machine with installed docker
   ```
    make network up php
    composer i
    ```

That's it. 

## APIs
You can find the generated documentation under `http://api.getrito.com.loc/api/docs`.
